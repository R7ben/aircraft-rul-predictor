# Screenshots for README

Drop 5 PNG files in this folder — the README uses them by exact filename:

| Filename | What to capture |
|---|---|
| `hero.png` | Wide shot of the Single Engine page — full browser window, cockpit visible |
| `single-engine.png` | Just the Single Engine page (browser tab content only) |
| `fleet-batch.png` | Fleet Batch page after uploading a CSV, KPIs + histogram visible |
| `compare.png` | Compare Engines page with two gauges filled in |
| `why.png` | Why? page showing the feature-importance chart |

## How to take clean screenshots

**Windows:** press `Win + Shift + S` → drag to select region → paste into any
image editor → save as `.png` with the exact filename above.

**Recommended browser setup before screenshotting:**

1. Full-screen the browser (`F11`).
2. Zoom to 90% (`Ctrl + -`) so the cockpit fits nicely.
3. Collapse Streamlit's "hamburger" menu (top-right of the app).
4. Refresh once so animations start clean.

## Tip: use browser DevTools for pixel-perfect

`Ctrl + Shift + I` → device toolbar → set a fixed resolution like 1440×900 →
right-click → "Capture full size screenshot". This gives you consistent
dimensions across all 5 screenshots.
